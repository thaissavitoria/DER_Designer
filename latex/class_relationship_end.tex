\doxysubsection{Detailed Description}
Represents one end of a relationship in an ERD diagram. 

This class manages the connection between a relationship and an entity, encapsulating the constraints and properties of that connection. It provides\+:
\begin{DoxyItemize}
\item \doxylink{class_entity}{Entity} identification through ID reference
\item \doxylink{_relationship_end_8h_a8926bbad0c6fb22d1c85cb4fdcd286d5}{Cardinality} constraints (One or Many)
\item Participation constraints (total or partial)
\item Custom text for cardinality display ("{}\+N"{}, "{}\+M"{}, ...)
\item String conversion utilities for serialization
\item Change notification system through Qt signals
\end{DoxyItemize}

Each relationship can have multiple ends, typically two (binary relationship) or more (n-\/ary relationship), connecting different entities with their respective constraints following Peter Chen\textquotesingle{}s ERD notation.
