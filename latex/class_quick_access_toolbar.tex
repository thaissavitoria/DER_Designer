\doxysubsection{Detailed Description}
Quick access toolbar for some ERD diagram operations. 

This class provides a customizable toolbar with quick access to some frequently used actions in the ERD Designer application. It extends QTool\+Bar to provide\+:
\begin{DoxyItemize}
\item Quick access to common file operations (New, Open, Save)
\item Help and About actions for user assistance
\item Integration with \doxylink{class_main_window}{Main\+Window} for action execution 
\end{DoxyItemize}
