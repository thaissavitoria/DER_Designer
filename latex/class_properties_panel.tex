\doxysubsection{Detailed Description}
Panel widget for viewing and editing properties of selected diagram elements. 

Displays a tree-\/based property editor that updates based on the current selection in the \doxylink{class_diagram_scene}{Diagram\+Scene}, supporting elements such as entities, relationships, and attributes, as well as connection lines.
