\doxysubsection{Detailed Description}
Represents a connection point on an ERD element for line attachment. 

This class manages connection points that serve as attachment locations for connection lines between ERD elements. It provides\+:
\begin{DoxyItemize}
\item Unique identification through UUID for tracking connections
\item Relative positioning within element boundaries (0.\+0 to 1.\+0 coordinates)
\item Direction specification for line orientation (Top, Bottom, Left, Right)
\item Absolute position calculation based on element position and size
\item Factory method for creating default connection points (4 cardinal directions)
\item Change notification system through Qt signals 
\end{DoxyItemize}
