\doxysubsection{Detailed Description}
Manages automatic saving and backup versioning of diagrams. 

Encapsulates all auto-\/save logic, including\+:
\begin{DoxyItemize}
\item Periodic saving to the current file
\item Versioned backup files with timestamps
\item Enable/disable control 
\end{DoxyItemize}
