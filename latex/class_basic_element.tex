\doxysubsection{Detailed Description}
Abstract base class for all ERD elements (Model only). 

This class provides\+:
\begin{DoxyItemize}
\item Basic properties common to all elements
\item Change notification system (Observer Pattern)
\item Serialization/\+Deserialization – to facilitate conversion to JSON
\item Connection points for line connecting mechanism 
\end{DoxyItemize}
