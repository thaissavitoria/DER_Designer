\doxysubsection{Detailed Description}
Interactive control point for Bezier curve manipulation in connections. 

This class provides a visual and interactive control point for adjusting Bezier curves in connection lines. It extends QGraphics\+Ellipse\+Item to provide\+:
\begin{DoxyItemize}
\item Visual representation as a draggable ellipse handle
\item Mouse interaction for dragging and repositioning
\item Hover effects for visual feedback
\item Communication with parent \doxylink{class_connection_graphics_item}{Connection\+Graphics\+Item} to update curve shape
\item Type identification (Control1 or Control2) for proper curve calculation 
\end{DoxyItemize}
