Detailed Description

Represents an entity element in an ERD diagram.

This class extends BasicElement to represent entities in Entity-Relationship diagrams following Peter Chen's notation. It provides:

  • Support for associated attributes through attribute IDs (from BasicElement)
  • Serialization for persistence
  • Type identification for rendering and behavior
